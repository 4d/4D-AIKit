# OpenAIFilesParameters

The `OpenAIFilesParameters` class contains parameters for creating files.

## Properties

| Property Name | Type                          | Description                          |
|---------------|-------------------------------|--------------------------------------|
| `file`        | [4D.File](https://developer.4d.com/docs/API/FileClass) | The File object to be uploaded.     |
| `purpose`     | Text                          | The intended purpose of the file.    |

Plus all properties from [OpenAIParameters](OpenAIParameters.md).

## Class constructor

Create a new instance with the specified parameters.

| Argument Name | Type   | Description                      |
|---------------|--------|----------------------------------|
| `object`      | Object | Object containing the parameters |
