# OpenAIAssistantDeleted

The `OpenAIAssistantDeleted` class represents the deletion status of an assistant in the OpenAI API.

## Properties

| Property Name | Type    | Description                                                      |
|---------------|---------|------------------------------------------------------------------|
| `id`          | Text    | The identifier of the deleted assistant.                        |
| `object`      | Text    | The object type, which is always "assistant.deleted".           |
| `deleted`     | Boolean | Whether the assistant was successfully deleted.                 |
